douRex - 豆瓣电台Opera扩展

douban, opera, extension

Download this project as a .zip file Download this project as a tar.gz file

douRex is a douban Radio extension for Opera.

Notes

  • 首次使用需要进入flash安全设置，将widget://加为可信路径，否则无法使用
  • 每次使用需要登录douban.fm获取cookie，否则可以正常收听，但加心和禁播操作不会记录到豆瓣
    (只登录douban.com无效，与Opera API有关，还没找到解决方法)

Features

  • [豆瓣电台]收听、暂停、换台、加心、禁播、切歌
  • 分享到豆瓣、饭否、新浪
  • 在专辑页面可选择从douRex播放
  • 在收藏页面可选择从douRex播放
  • 去g.cn搜歌词
